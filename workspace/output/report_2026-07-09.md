## 6. 가정 목록

- 입력 방식: 대화형(CSV 없음)
- 전체 사용자 수: 4,000명
- 평일 일간 활성 사용자 수(Weekday DAU): 400명
- 동시접속률: 10% 가정(DAU 400명 중 약 40명 동시 접속) — 별도 확인 필요
- 평균 트랜잭션/세션: 법인카드 결제 내역 조회·전표 처리 기준 경량 트랜잭션 가정
- Object Store 사용량: 영수증/전자결의서 첨부파일 기준 약 100GB로 가정(실측 필요)
- HANA Cloud는 Small 티어로 충분하다고 가정(트랜잭션/데이터량 기준 재무 시스템 대비 경량)
- 월 730시간(24시간 x 30.4일) 기준으로 시간당 과금 자원의 월 비용 환산
